GeoEarth

Javascript library to visualize GeoJson on a globe

Interactive

Play with your 3D geographic data.

Web Based

Use Javascript to visualize a globe in the browser.

Open Source

Hack it, break it, improve it.

Get Started 

var opts = {
  colorFn: 0xff0000,
  textureImage: './textures/earth-black.jpg',
  earthRadius: 160,
  // disableAtmosphere: true
  // autoStart: false
};
var container = document.getElementById("geo-container");
var geoearth = new GeoEarth(container, opts);
geoearth.addPoint( [72.877426, 19.076090] , {size: 4} );
